「Ctrl + G」行数跳转

「Ctrl + E」当前文件内按「Ctrl + E」,输入文件名可快速打开其他文件

「F2」重命名一个变量：选中一个变量，然后按F2，弹出一个小窗口，在里面输入内容后按回车，所有该变量都会被重命名。

「Ctrl + Alt + UP/DOWN」可以使一列上出现多个光标

「Shift + Alt + MOUSE」可以出现竖直的列光标，同时可以选中多列

「Shift + Alt + I」,在选中的段文字每行结尾添加光标

「Ctrl + Shift + L」或者「Ctrl + F2」，可以在页面中出现这个词的不同地方
    都出现光标。有时候这个快捷键的作用和f2重命名变量类似，但是它更加广泛，因为
    还可以对比如字符串相同的非同一变量或函数类的东西修改。

「Shift + Alt + MOUST」可以出现竖直的列光标，同时可以选中多列。

    任何光标操作，可以按Ctrl + U取消
